# 金沢データセット追加学習（パターンA）報告書

## 1. 実験目的
CARIM (Context-Aware Retrieval with Video Playback) モデルに対して、新たに「金沢データセット (Kanazawa Dataset)」専用のドメイン特化型適応（ファインチューニング）を行うことを目的としました。オリジナルのNuScenesデータセットで学習済みの重みを初期値として用いることで、金沢特有の街並みや天候、時間帯などの環境に即した特徴表現を獲得（パターンA）し、専用の検索アプリケーションでより精度の高いコンテキスト検索を可能にすることを目指しました。

## 2. 実験条件・概要

### 2.1. データセットとパイプライン
- **対象データセット**: 金沢データセット（`20250127_151151`）
- **総画像数**: 30,092 枚 (10 FPS換算で約50分に相当する連続フレーム)
- **学習パイプラインの手順**:
  1. **ベースデータセット作成**: 画像パス一覧の構築（`train.jsonl`の生成）
  2. **画像キャプション生成**: 大規模視覚言語モデル（VLM）を用いた、各画像に対する詳細な情景説明（キャプション）の付与。
  3. **要素抽出（Refinement）**: 生成されたキャプションから、検索に有効なキーフレーズ（天候、時間、オブジェクト等）をLLMで抽出。
  4. **データセットマージ**: キャプションと抽出要素を統合した最終的な学習用データ（`train_full.jsonl`）の構築。
  5. **追加学習（対照学習）**: Element-Guided Contrastive Learning と Adaptive Negative Injection (ANI) を利用した特徴表現学習。

### 2.2. 使用モデルとパラメータ
- **キャプション生成モデル**: `Qwen/Qwen-VL-Chat`
- **要素抽出 (Refinement) モデル**: `Qwen/Qwen2-1.5B-Instruct`
- **CARIM (Text-to-Text) エンコーダ**: `Qwen/Qwen2-1.5B-Instruct`
- **ベース重み**: NuScenesデータセットで事前学習された `carim_train_intermediate.pt` (CARIM v1.0)
- **学習Epoch数**: 5 Epochs
- **Batch Size**: 24

### 2.3. 計算機環境と所要時間
- **使用ノード/GPU**: 計算クラスター（パーティション: `a6000_ada`, `a6000`）、GPU: NVIDIA RTX 6000 Ada (1基)
- **各処理の所要時間**:
  - **キャプション生成**: 約 65～70時間 （※24時間でタイムアウトし、約11,000枚処理したのち7日間上限で再開完了）
  - **要素抽出 (Refinement)**: 約 2.5時間
  - **対照学習（ファインチューニング）**: 1 Epochあたり約1時間。5 Epoch合計で **約5.2時間**
  - **検索インデックス生成 (30,092枚分)**: 約15分

## 3. 実験結果

### 3.1. 学習の推移
学習中に算出される Average Loss は以下のように順調に減少・収束しました。
- Epoch 1: `Avg Loss=0.9850` 未満からのスタート
- Epoch 4: `Avg Loss=0.6939`
- 最終的な学習済み重み: `carim_kanazawa_finetuned.pt` (約 6.1 GB)

ロス内の詳細として、Contrastive Loss は 0.2前後、Negative/Positive Penalty Loss も安定した値（0.15~0.25）で推移しており、崩壊することなく良好な特徴空間マップの構築に成功しています。

### 3.2. 検索アプリケーションでの評価
生成された特徴量インデックス（`text_index.pt`）を用いて、金沢専用のCARIM検索アプリケーション（`kanazawa_ver/app.py`）を起動しました。
- 単一の巨大なシーンとして誤認識されないよう、Kanazawaデータのタイムスタンプを利用して **「約300フレーム（約30秒）ごとの仮想シーン」** に動的に分割するアルゴリズムを導入しました。
- これにより、クエリに対する該当フレームの検索だけでなく、前後のコンテキスト映像を即座にUI上の動画として再生・確認することが可能となりました。
- NuScenesでは捉えきれなかったかもしれない「金沢特有の街並み」に対する細かな言語指示にも、より強力に反応できるようになっています。
